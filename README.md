# Nook Calendar Code Assessment

This is a basic skeleton app built in [React Native](https://reactnative.dev/) that serves the user a joke setup through a push notification after clicking a button in a drawer.
Upon clicking the notification, the app navigates to a new page and shows the punchline.

## Installation 

Clone this project to a desired local folder. 

```
$ git clone https://github.com/fnyah/nook-assessment.git
$ cd nook-assessment
```

Install project dependencies. 

```
$ npm install
```

Run the app!

```
npm start
```

# Required Dependencies

* [Expo](https://expo.dev/)
* [React Navigation](https://reactnavigation.org/)
